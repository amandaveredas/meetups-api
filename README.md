# AGENDADOR DE MEETUPS

O objetivo desse serviço é possibilitar o agendamento de encontros online/presenciais.

Funcionalidades na V.0.1:

* Criar um registro
* Alterar um registro
* Buscar um registro
* Buscar todos os registros
* Filtrar registros
* Excluir um registro
* Criar um meetup
* Alterar um meetup
* Buscar um meetup
* Buscar todos os meetups
* Filtrar meetups
* Excluir uma meetup
* Cadastrar registro(s) em uma meetup
